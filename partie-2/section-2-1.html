Introduction à l'Ingénierie des Données

Introduction

Bienvenue dans le cours "Création de pipelines de données par lots sur Google Cloud".

Je suis Damon, développeur de contenus pédagogiques techniques chez Google.

Ce cours constitue le deuxième module de la série « Ingénierie des données sur Google Cloud ».

Dans le module précédent, nous avons abordé les lacs de données et les entrepôts de données comme options de stockage pour vos données.

Dans ce cours, nous allons nous concentrer sur la création de pipelines de données par lots permettant de charger les données dans des systèmes de stockage, en utilisant les méthodes suivantes :

  • Extract Load (EL)
  • Extract Transform Load (ETL)
  • Extract Load Transform (ELT)

Nous étudierons plusieurs technologies de transformation des données sur Google Cloud, dont :

  • BigQuery
  • Exécution de Spark sur Dataproc
  • Traitement de données sans serveur avec Dataflow
  • Utilisation de Cloud Data Fusion et Cloud Composer pour les pipelines
