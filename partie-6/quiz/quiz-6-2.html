🔎 Écosystème Power BI – Questions/Réponses

📌 Qu’est-ce que Power BI Desktop ?

C’est l’outil de développement local pour créer des rapports Power BI. Il permet de se connecter aux données, les transformer avec Power Query, modéliser avec DAX et relations, puis construire des rapports interactifs.
💡 Tout projet Power BI commence ici.

☁️ À quoi sert Power BI Service (app.powerbi.com) ?

Il s’agit de la plateforme cloud de publication, de collaboration et de diffusion. Elle permet de :

  • Publier les rapports depuis Power BI Desktop
  • Gérer les workspaces (projets, accès, rôles)
  • Configurer les refreshs automatiques
  • Partager les rapports avec les utilisateurs
  • Créer des dashboards à partir de plusieurs rapports
  • Superviser la sécurité, l’usage et la performance

🔄 À quoi sert un Power BI Dataflow ?

C’est un outil d’ETL dans le cloud via Power Query Online. Il permet de centraliser les transformations de données, de mutualiser les sources et de créer des tables propres réutilisables dans plusieurs datasets.

📊 Qu’est-ce qu’un Power BI Dataset ?

C’est le modèle de données central dans Power BI. Il contient les tables, relations, mesures DAX, sécurité RLS, etc. Il peut être partagé entre plusieurs rapports.
🎯 C’est le cœur du reporting analytique.

🛡️ À quoi sert Power BI Gateway ?

Il s’agit d’une passerelle installée en local pour permettre à Power BI Service d’accéder à des données internes (non cloud) comme SQL Server, Oracle, fichiers Excel sur serveur, etc.
⚠️ Indispensable pour les sources on-premise.

📄 Quand utiliser Power BI Report Builder ?

Pour créer des rapports paginés au format PDF, comme des factures, listes de commandes ou documents à imprimer. Il est principalement utilisé avec Power BI Premium.

📱 Que permet Power BI Mobile ?

C’est l’application mobile (iOS/Android) pour consulter les rapports et dashboards en mobilité. L’interface est adaptée aux écrans de smartphones et tablettes.

🔗 Qu’est-ce que Power BI Embedded ?

C’est une solution pour intégrer les visuels Power BI dans des applications web, portails clients ou outils métiers. Elle fonctionne via des API et des tokens d’authentification.

⚙️ Quel est le rôle de Power BI REST API et de l’Admin Portal ?

Ils permettent l’administration avancée et l’automatisation de Power BI à l’échelle de l’entreprise : gestion des workspaces, utilisateurs, actualisations, monitoring, etc., via scripts ou outils externes.
