Écosystème Power BI

  1. Power BI Desktop
    Outil de développement local
    Permet de se connecter aux données, transformer (Power Query), modéliser (relations, DAX) et créer des rapports interactifs. C’est ici que commence tout projet Power BI.
  2. Power BI Service (app.powerbi.com)
    Plateforme cloud de publication et de partage
    C’est l’environnement en ligne pour :
    • Publier les rapports créés dans Desktop
    • Gérer les workspaces (projets/collaboration)
    • Programmer les refreshs
    • Partager avec les utilisateurs
    • Créer des dashboards à partir de rapports
    • Superviser les usages et la sécurité
  3. Power BI Dataflows
    ETL dans le cloud (Power Query Online)
    Permet de centraliser les transformations de données, mutualiser les sources, créer des tables propres avant qu’elles soient utilisées dans plusieurs datasets.
  4. Power BI Datasets
    Modèle de données central
    Contient les tables, relations, mesures DAX, sécurité RLS, etc. Peut être partagé entre plusieurs rapports. Le dataset est le cœur du reporting analytique.
  5. Power BI Gateway
    Passerelle de connexion aux données on-premise
    Installée localement, elle permet à Power BI Service d’accéder à des données internes (SQL Server, Oracle, Excel local…). Obligatoire pour les sources non cloud.
  6. Power BI Report Builder
    Création de rapports paginés
    Idéal pour les états de type PDF à imprimer ou exporter (factures, listes détaillées…). Utilisé principalement dans Power BI Premium.
  7. Power BI Mobile
    Application mobile (iOS, Android)
    Permet de consulter les rapports et dashboards en mobilité, avec une ergonomie adaptée aux smartphones/tablettes.
  8. Power BI Embedded
    Intégration de rapports dans des applications tierces
    Permet d’embarquer des visuels Power BI dans des portails web, des applis métiers ou clients, via API et token.
  9. Power BI REST API / Admin Portal
    Administration et automatisation
    Utilisé pour gérer les workspaces, utilisateurs, refreshs, etc., via scripts ou outils externes. Très utile en entreprise pour industrialiser.
