Quiz : Qualité des Données et Traitement dans BigQuery

1. Quelle est la principale raison pour laquelle nous traitons la qualité des données ?

  • a) Pour augmenter la taille des données
  • b) Pour améliorer les performances du stockage
  • c) Pour éviter que des données de mauvaise qualité nuisent à l'analyse et aux décisions commerciales
  • d) Pour rendre les données plus faciles à stocker

2. Quelle est une des caractéristiques de données qui peut être **invalide** ?

  • a) Une transaction de 7$ au lieu de 10$ pour un ticket de cinéma
  • b) Une information qui n'est pas en format texte
  • c) Une donnée en double dans une table
  • d) Une donnée manquante

3. Quelle est la différence entre **validité** et **précision** des données ?

  • a) La validité concerne les erreurs humaines, tandis que la précision concerne les erreurs techniques

  • c) La validité se réfère à la quantité de données, tandis que la précision concerne la rapidité de l'accès
  • d) La validité et la précision sont synonymes dans ce contexte

4. Comment BigQuery peut-il résoudre le problème des **doublons** dans les données ?

  • a) En supprimant automatiquement les doublons lors de l'importation
  • b) En utilisant la fonction `COUNT DISTINCT` pour calculer les résultats uniques sans avoir à transformer les données
  • c) En utilisant des vues pour stocker les doublons et les gérer plus facilement
  • d) En créant un script Python pour nettoyer les doublons

5. Dans quel cas **BigQuery View** peut-il être utile ?

  • a) Pour créer un ensemble de données de test
  • b) Pour permettre aux utilisateurs d'accéder à des données nettoyées et filtrées sans affecter les données brutes
  • c) Pour faire des calculs de données complexes
  • d) Pour créer une copie des données d'origine dans une autre table

6. Que signifie **complétude** des données ?

  • a) Les données sont exactes et précises
  • b) Toutes les informations nécessaires sont présentes dans les ensembles de données
  • c) Les données sont cohérentes dans toutes les sources
